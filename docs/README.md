# docs/

Extended documentation for astroCAMP-bench.

Suggested contents:

- `protocol_v0.1.md` – astroCAMP community benchmark protocol (goals, tasks, rules)
- `metrics_spec.md` – detailed metric definitions and formulas
- `datasets.md` – dataset description and download instructions
- `baselines.md` – description of baseline implementations
- `contributing.md` – guidelines for new tasks, metrics, and submissions

The main paper text can be mirrored here as a high-level overview, while this
directory provides operational details for users and contributors running the
benchmark.
