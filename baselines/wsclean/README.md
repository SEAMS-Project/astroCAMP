# baselines/wsclean/

Baseline benchmark pipeline using WSClean (and optional IDG integration).

Suggested contents:

- `run_C1_wsclean.sh` – run script for C1 closed/open track
- `run_C2_wsclean.sh` – run script for C2
- `configs/` – WSClean-specific configs used in the benchmark
- `logs/` – example logs and metric outputs

Use this folder as a template for contributors adding new WSClean-based
submissions or optimised variants.
