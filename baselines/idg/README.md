# baselines/idg/

Baseline benchmark pipeline using IDG and WS-Snapshot or related algorithms.

Suggested contents:

- `run_C1_idg.sh` – run script for C1 with IDG
- `configs/` – IDG-specific configs and gridder settings
- `logs/` – example logs and metric outputs

This folder illustrates how to integrate IDG and snapshotting strategies
into the astroCAMP benchmark protocol.
