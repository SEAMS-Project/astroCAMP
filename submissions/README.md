# submissions/

Example submission packages and templates for astroCAMP-bench.

Each submission should contain:

- `system.json` – hardware and software description
- `config.yaml` – benchmark configuration
- `metrics.json` – computed metrics (runtime, energy, quality, etc.)
- optional log files or plots

Use this directory to document the expected layout of community contributions,
and to store reference submissions mentioned in the astroCAMP paper.
